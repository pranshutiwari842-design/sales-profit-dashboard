# Sales and Profit Dashboard

## Project Overview
This project is an interactive Sales and Profit Dashboard created using Microsoft Excel. The dashboard helps analyze business performance by visualizing sales, profit, and regional trends in an easy-to-understand format.

## Features
- Sales Performance Analysis
- Profit Trend Visualization
- Region-wise Analysis
- Category-wise Insights
- Interactive Charts and Pivot Tables
- Business Decision Support

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning
- Data Visualization
